Les differents liens avec l'attribut target

  1. _self: est la valeur par défaut: la page cible s'ouvre dans le même emplacement que là où l'utilisateur a cliqué.
    2. wikipédia s'ouvre au même endroit
  2. _blank: la page cible s'ouvre dans un nouvel onglet ou dans une nouvelle fenêtre
    3. wikipedias'ouvre dans un nouvel onglet
  3. _parent: la page cible s'ouvre dans le cadre (frame) de niveau immédiatement supérieur par rapport à l'emplacement du lien
    4. wikipedia s'ouvre au niveau immédiatement supérieur
  4. _top: la page cible s'ouvre dans la fenêtre hôte (par dessus le frameset)
    5. wikipedias'ouvre dans la fenêtre hôte
